# Dark Matter Studio — Pricing Guide

All prices in USD. Fixed-fee per project. No hidden costs, no hourly billing.

## How I price

Each project is quoted as a fixed fee based on scope. Scope drives price; effort doesn't.
The same number of hours on a 3-page portfolio and a multi-page corporate site produce
very different outcomes, so they're priced differently regardless of effort.

I don't negotiate price after a proposal is sent. Scope adjustments happen up front;
once the proposal is signed, the price is the price.

## Project ranges

### Simple portfolio / 1-3 page static site
**$500 - $1,200**

For solo creatives and small businesses needing a credible online presence.
Single-page or 2-3 simple pages. Static content, minimal interaction.
Example reference: Sussex Light Photography.

### Standard landing page
**$800 - $2,500**

Conversion-focused single-page site. Includes responsive design, contact/inquiry
form, basic SEO setup, light animation. For service businesses driving traffic
to one core offering.

### Direct booking / single-product site
**$1,500 - $3,500**

Single-page rich-content site for one bookable product or service. Includes
detailed sections, gallery, testimonials, transparent pricing, inquiry form.
Example reference: The Crib Murree.

### Multi-page corporate site
**$2,500 - $6,000**

Information architecture across multiple categories, navigation, sub-pages.
For professional services firms, agencies, or businesses needing a full
marketing presence. Includes integrations (booking, chat, analytics).
Example reference: FMY Chartered Accountants.

### SaaS marketing site
**$2,000 - $5,000**

Multi-section marketing site for software products. Hero, features, pricing,
testimonials, FAQ, signup CTAs. Built for conversion and clarity.

### Redesign of existing site
**60-75% of equivalent new-build pricing**

When existing brand assets and content can be carried over, redesigns cost
less than greenfield builds. New brand/IA work pushes back toward full price.

## What pushes a project to the top of its range

- Multiple stakeholders or decision-makers in the review process
- Tight deadlines (under 3 weeks for substantial work)
- Custom animations, illustrations, or complex interactions
- Third-party integrations beyond standard (CRMs, payment systems, custom APIs)
- Content strategy and copywriting included
- Heavy revision cycles requested
- E-commerce or auth flows

## What keeps a project at the bottom of its range

- Single decision-maker, fast feedback
- Existing brand guidelines and assets ready to go
- Generous timeline (4+ weeks)
- Standard layouts with minor customization
- Client provides finalized copy
- No third-party integrations beyond basic forms

## Add-ons (separate from project price)

These are not modifications to the base project price — they're additional
services billed separately.

- **Quarterly performance review:** $200/quarter — analytics review, performance audit, recommendations
- **Ongoing support retainer:** $300-$600/month — minor updates, bug fixes, content changes
- **SEO content strategy:** $500-$1,500 — keyword research, content plan, on-page implementation
- **Additional design rounds beyond two:** $200/round

## Quick reference for proposals

When a lead describes a project, anchor pricing by:
1. Identifying which project type fits best (portfolio, landing page, corporate site, etc.)
2. Picking the middle of the range as the default
3. Adjusting up if the scope-drivers above apply
4. Adjusting down if the scope-keepers apply

Quote a single number, not a range, when sending the actual proposal.
Ranges go in conversations; firm numbers go in writing.
